# More Ui-changes

## Board

The board needs more work for it to convey everythig to the player accurately

### Borders

The borders are placed correctly but i have a tiny nitpick about where the two colors meet each other
If possible have the borders meet in a clean way where it doesnt look like unintentional overlap

### planning

planning more is great, i also like the translucent look of planned moves
However to convey more meaning, planning stones should also swap color with every click to reflect the players taking turns

### arrows

planning arrows are fine as-is but it would be great if they could be rendered while being drawn

### gaps

The board features nice gaps that set individual spaces apart
However when placing stones on the border that gap cannot be seen

### Winning connection

highlighting the winning connection has some z-issues (i think)

## More UI

Resigning and then refreshing the game will should put you back in the lobby

### Animations

The switching animations are nice, but i notice some bouncing in the animation. If this is trust id rather have an exponential curve 

### Clock

When creating an invite link the clock settings use standard up/down arrows, a better way would be to have +- buttons
or just styled buttons

### engine selection

the field for engine selection should only take up half the available space instead of all
The field also shows a misleading dropdown icon
The popup itself is nice but can also do with some touch ups

* Radio button is not needed the outline already shows the active selection
* pressing esc should close the menu
* the separator line of engine name and dropdown icon is not needed
* The drop down icon can be reworked to use a nice V - shaped arrow with a gentle angle

### waiting for opponent after invite link

Also when creating the match copy the link to clipboard for good measure
Cancel button should have a confirmation dialogue, and the button should be red using the same design
