# UI-design 2

The Colors are nice but a bit samey, the design could do with a little more contrast
The glow around some buttons is obnoxious and distracting
Gradient filled buttons are also not great, i like the 'hollow' buttons more

## Rule list

Move the elements of the rule selection to be in this order

* Board size element without 'Board size' text
* Play as [aligned left]; Swap rule [aligned right]
* Engine button without explanatory text

No unlimited time text for the engine settings

When hovering the text (not selection) for Play as and swap rule an info popup shows

* For play as: which color goes first
* for Swap rule: explain the swap rule in one sentence

## Swap button

The rule toggle that allows/disallows swap has a broken animation

## board size and other radio buttons (play as, ...)

Big fan of the design from the color selection button, can you apply this to the board size selection
If possible, an animation that plays when the selection is changed would be cool, the color highlighting the active selection swooshes to the clicked target

## Queue

The buttons for queuing should have a nice ratio, either 4:3, golden ratio or square

## Engine selection

The drop down for engine selection is a little squished, can you create a popup that lists the engines in a scrollable list of engines?
The list should have a single selection harmonica, where an arrow down on the right of each toggles advanced egine information.
Then, if this works, the currently selected engine is displayed in a button that brings up the selection dialog on click
